# **App Name**: BHA Book Payment Tracker

## Core Features:

- Data Table Display: Display student names and payment statuses in a tabular format, with sortable columns.
- Payment Status Input: Allow manual input of payment dates for each of the three payment deadlines per student.
- Automated Payment Tracking: Automatically calculate payments, balances, and outstanding amounts.
- Summary Dashboard: Provide an overview dashboard showing total amount due, amount received, and remaining balance, automatically calculated.
- Data Export: Enable exporting data to CSV format for offline reporting.
- Student Search: Enable quick search and filter of student records.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and reliability.
- Background color: Very light blue (#E8EAF6), near-white but softer and visually distinct from white.
- Accent color: Muted green (#7CB342) to highlight positive payment status and important actions.
- Body and headline font: 'Inter', a sans-serif font that ensures readability and a clean, modern appearance.
- Use clear, simple icons to represent payment statuses and actions.
- Maintain a clean, organized layout with clear sections for data entry and summaries.
- Subtle animations, such as highlighting changed cells, will give visual feedback and create a polished, user-friendly experience
